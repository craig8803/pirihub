# PiriHub

A website for coliving in Pirenópolis, Brazil.

## Pages

- **Home**: Introduction to PiriHub
- **Coliving**: Information about coliving lifestyle
- **Houses**: Dropdown menu to select and view details of four houses
- **Services**: List of available services

## Structure

- `index.html`: Home page
- `coliving.html`: Coliving page
- `houses.html`: Houses page with dropdown
- `services.html`: Services page
- `css/styles.css`: Stylesheet
- `js/script.js`: JavaScript for interactivity
- `img/`: Directory for images

## Usage

Open `index.html` in a web browser to view the site.

## Troubleshooting

- Ensure all files are in the same directory structure.
- If JavaScript doesn't work, check browser console for errors.
